# Vibe PPT

바이브코딩으로 만든 웹 서비스들 - 발표용 웹 프레젠테이션

## Overview

Reveal.js 기반 웹 PPT. AI 음악 SaaS, 비디오 생성기, 제너레이티브 아트, 웹 3D 인터랙티브 프로젝트를 소개하는 16섹션(세로 서브슬라이드 포함) 발표 자료.

## Tech Stack

- **Reveal.js** - 프레젠테이션 프레임워크
- **Vanilla JS** - 커스텀 페이지네이션, 파티클 이펙트, iframe lazy loading
- **CSS** - Editorial botanical 테마 (Playfair Display + Pretendard Variable)

## Features

- 커스텀 페이지네이션 (prev/next + dots + sub-dots)
- 자동 순차 애니메이션 (.seq 클래스)
- 보태니컬 캔버스 파티클 이펙트
- iframe 데모 lazy loading
- 스피커 노트 지원 (S키)
- URL 해시 기반 슬라이드 위치 유지
